Get set up in 3 minutes
01Go to console.cloud.google.com and sign in with your Google account.
02Create a new project (or use an existing one). Click Enable APIs → search Places API → Enable it.
03Go to Credentials → Create Credentials → API Key. Copy the key.
04Add a billing account (required even for free tier — Google won't charge under $200/mo). This entire job costs under $1.
05Paste your key in the left panel, select trades + cities, hit Run search. Results load in ~30 seconds.
Important: This tool calls the Google Places API directly from your browser. Your API key is never sent anywhere except Google. Results are stored locally in your browser — nothing leaves your machine except the API calls.
